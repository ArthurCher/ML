В исследовании используется прямое скачивание данных с kaggle.com
Для работы данного скрипта необходимо:
- Зарегистрировать на kaggle.com
- Установить утилиту kaggle запустив команду:
pip3 install kaggle
- Получить токен и подключить kaggle api, инструкция по ссылке https://www.kaggle.com/docs/api

Также можно получить данные в папке /data в файле winequalityN.csv
